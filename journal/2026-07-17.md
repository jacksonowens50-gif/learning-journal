## Explaining Git
- Git is how we will track all changes for your project, image it as versions of an excel file. When an analyst updates an excel file from v1 to v2 to v3, we can track all the changes between them and revert back to a version if needed. Git tracks all these with comments so that the reason for changes is never lost and we can fall back on older versions if needed.
